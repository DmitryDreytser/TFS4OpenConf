
Скрипт предназначен для проверки внутренних идентификаторов метаданных на принадлежность заданному диапазону.
Может использоваться для разрешения конфликтов внутр. идентификаторов при одновременной работе нескольких 
разработчиков. Для этого в репозитарий помещается текстовый файл, в котором администратор проекта для
каждого разработчика добавляет строки вида
<ИмяПользователя> <min. ID> <max. ID>
На каждого разработчика может быть по нескольку строк в файле - берётся всегда самая последняя. 


Скрипт написан в 2-х вариантах: на Perl'е и на Python'е.

Для получения списка параметров скрипта нужно выполнить команду
\>perl check_max_id.pl -h
или
\>perl check_max_id.py -h
