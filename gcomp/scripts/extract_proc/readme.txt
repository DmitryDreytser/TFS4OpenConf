ExtractProc.py - разбирает на процедуры программные модули конфигурации 1С (файлы *.1s).

Основное назначение - исследование и слияние изменений в больших модулях.

Разбирает указанный программный модуль ('<ПутьДоМодуля>\<ИмяМодуля>.1s') на
процедуры. Каждая процедура помещается в файл с именем
'<ИмяВыходногоКаталога>\<ИмяПроцедуры>.txt', а сам модуль в файл
'<ИмяВыходногоКаталога>\<ИмяМодуля>.1ss'.

По умолчанию <ИмяВыходногоКаталога> совпадает с '<ПутьДоМодуля>\<ИмяМодуля>'.

Дополнительно можно разобрать все программные модули в указаном каталоге и всех 
подкаталогах. Имена файлов образуются аналогично.

Опции командной строки и примеры использования можно посмотреть запустив скрипт
с ключом -h и -E соответственно.
