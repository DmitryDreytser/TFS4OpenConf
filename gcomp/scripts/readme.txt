Здесь лежат самые разнообразные скрипты, с помощью которых можно
существенно расширить функционал gcomp'а.

ОГЛАВЛЕНИЕ
================================================================
auto_add....................Автоматическое выполнение команд 
                            cvs add и cvs remove на основании
                            лога gcomp -d -v.
check_max_id................Сверка текущего ID метаданных с
                            заданным диапазоном.
code_beautifyer.............Автоформатирование кода 1С.
correct_dlg.................Корректор сместившихся элементов в 
                            диалоговых формах.
extract_proc................Разборка модулей на процедуры - каждая 
                            процедура в отдельный файл.
repository_cvt..............Конвертор репозитариев, созданных
                            старыми версиями gcomp.
show_pics...................Помошник в поиске картинок на формах 
							конфигурации.
Code_Works..................Скрипты для манипуляций с кодом (вставка 
                            процедур, замена кода, и т.п.)							
Metadata_works..............Скрипты для работы с метаданными							

Пока тут больше ничего нет. Желающие разместить свои скрипты -
присылайте их на fe@alterplast.ru, в формате, близком к тому, что 
есть в show_pics. То есть, сам скрипт (или несколько), и ОБЯЗАТЕЛЬНО
описание того, ЧТО и КАК этот скрипт делает. adirks@ngs.ru
Адреса для обратной связи также приветствуются.
