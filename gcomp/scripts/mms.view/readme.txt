
Пример парсера потока Main Metadata Stream.
Наверное может пригодиться и для парсинга любых других 1С-вых форматов со скобками
(типа экранных диалогов).

Предполагается, что в каталоге, откуда запускается скрипт - находится файл с именем
Main Metadata Stream

Скрипт мало пригоден для реальных задач, он создавался только как пример легкого 
и быстрого парсинга MMS-подобных конструкций.
