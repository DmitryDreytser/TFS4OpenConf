Введение

Данный документ описывает структуру файлов "*.md/UserDef/Page.2/Page.*". Являющимся набором прав того, или иного пользователя.
Файл прав состоит из заголовка, в котором указывает количество блоков (наборов) прав идущих после. Каждый блок прав состоит из своего заголовка, в котором указывается, тип набора (например Справочник или Документ), его индетификационный код (Код МД) в потоке метаданных, и количества содержащихся в нем свойств. Если "Код МД"=0, то это общий набор для всех по умолчанию.
Краткие примечания:
  • Если в блоке прав не менялись значения по умолчанию, то данный блок не сохраняется. Кроме общего блока, он сохраняется всегда.
  • Если один раз изменить некий блок права, то запись о нем остается постоянно.
  • Новые данные добавляются в конец файла.
  • Код МД - идентификационный номер в потоке метаданных. Для общих прав он равен 0.
  • Количество в блоке свойств не обязательно постоянно.
